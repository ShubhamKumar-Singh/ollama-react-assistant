# React AI Chatbot with Ollama

This project is a React-based AI chatbot with a beautiful Material-UI interface, using Ollama for AI responses.

## Features
- Modern, responsive UI with Material-UI
- Real-time AI chat powered by Ollama (local API)
- TypeScript for best coding conventions

## Setup
1. Ensure Ollama is running locally (default: http://localhost:11434)
2. Install dependencies:
	```bash
	npm install
	```
3. Start the app:
	```bash
	npm start
	```

## Customization
- Change the model in `src/ollamaApi.ts` as needed (default: llama3.2)
- UI can be themed via Material-UI

## Coding Conventions
- Functional components with hooks
- TypeScript for type safety
- API utilities separated in `src/ollamaApi.ts`
- Clean, readable code structure
